# RAGent 测试文档

## 简介

这是一个用于测试文档解析、清洗和切分流程的示例文档。

## 功能特性

1. 多格式文档解析（PDF, DOCX, TXT, MD, 图片等）
2. 编码自动检测（UTF-8, GBK, GB2312）
3. 文本清洗（去除控制字符、页码标记、隐形字符）
4. 智能切分（512字符/块，64字符重叠）

## 技术架构

整个系统分为以下几个核心模块：

- 文档解析器：基于 Docling，支持多种文档格式
- 文本清洗器：去除噪音字符，标准化文本
- 文本分块器：将长文本切分为适合检索的片段
- 向量化引擎：调用 SiliconFlow API 生成文本嵌入
- 向量数据库：使用 ChromaDB 存储和检索

## 常见问题

问：支持哪些文件格式？
答：支持 PDF、DOCX、PPTX、XLSX、TXT、MD、CSV、HTML、JSON、XML 以及常见图片格式。

问：如何处理超大文档？
答：文档会被自动分块处理，每块 512 字符，支持并发处理多张图片。

## 总结

RAGent 是一个模块化的 RAG 文档处理系统，通过清晰的管道设计实现了文档解析、清洗、分块和向量化的全流程自动化。后续计划支持更多文档格式和优化策略。
